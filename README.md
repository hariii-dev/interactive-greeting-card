# 🥳 Interactive Birthday Greeting Card

A clean, responsive, and interactive digital Birthday Greeting Card built using pure HTML5 and CSS3. This project showcases smooth CSS transitions, transformations, and pseudo-elements.

## 🛠️ Tech Stack
- **HTML5** - Structured semantic layout.
- **CSS3** - Custom styling, flexbox layouts, hover effects, and 2D transformations.

## ✨ Key Features
- **Smooth Scaling:** The card gently scales up (`transform: scale(1.1)`) and changes background color when hovered.
- **Dynamic Content:** Uses CSS `::before` and `::after` pseudo-elements to inject emojis into the header.
- **Skew Effects:** The response sections feature a cool skew effect (`transform: skewX(10deg)`) on hover.
- **Accessible Links:** Fully styled interactive links with custom `:hover`, `:active`, and `:focus` states.
